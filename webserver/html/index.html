Demo Bioatlas Sweden

This is a landing page with links through to the components installed on this server. This is a test installation and consist of the basic ALA modules. The modules running on this server have been deployed as Docker containers. The source code for the installation can be found here.

Occurrence store

The occurrence store should now be setup. This application consists of 3 web applications :

  • Occurrence search - Reskinnable front end UI
  • Explore your area - Reskinnable front end UI
  • Apache SOLR - the index used for searching (note port 8983 needs to be open for this link to work)
There is also a command line tool now installed that can be started with the command "biocache". This tool can be used for loading data and running processes across the data.

Further information

As part of the installation the following tools are installed:

  • Apache Cassandra
Cassandra can be accessed directly using the command-line tool cqlsh.

Data registry

The Data registry (called the 'collectory') is installed. This is an application for editing metadata for collections, institutions and data resources.

  • Collections browser
  • Datasets browser
  • Web services
  • Admin interface

Further information

As part of the installation the following tools are installed:

  • MySQL
MySQL can be accessed directly using the command-line tool mysql.

Image Service

The image service is 1 web application interacting with a PostgreSQL database and Elasticsearch backend.

  • Image Service - Reskinnable front end UI

Bootstrap 3

The applications use Bootstrap 3 for layout and styling. For more information on Bootstrap see the Bootstrap documentation.

Styling these apps

These applications use a banner, footer and head html block that you can customise.

  • banner.html - the top section added to all pages. Typically a navigation menu
  • footer.html - the bottom section added to all pages. Typically a navigation menu
  • head.html - a block of html that is added to the <head/> section added to all pages. This could include references to custom CSS.

Species lists

The Species lists component consists of 1 web application :

  • Species lists - Reskinnable front end UI

Species pages

The Species pages (BIE) consists of 2 web applications :

  • BIE UI - Reskinnable front end UI
  • BIE web services - taxonomy search services

Name indexes

Lucene name indexes have been installed in the /data/lucene directory. Use the nameindexer tool and your own DwCA to generate a new index. See here for more details on the structure of the DwCA required.
